JavaScript is a dynamically typed language, meaning you don't need to declare a variable's type ahead of time. The types are determined during the execution.

You can re-assign a single variable to reference different types.

var VS let

Scoping: var is scoped to the immediate function body. let is scoped to the immediate enclosing block.

Hoisting: Variable (var) declarations are hoisted. let variable is not hoisted.

Re-declaration: var can be re-declared multiple times. let cannot.

const

Unlike in Java, const can not be left unitialized immediately: const a; will throw SyntaxError.
